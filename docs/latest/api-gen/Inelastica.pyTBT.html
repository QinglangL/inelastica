python TBTrans (Inelastica.pyTBT)¶

Magnus Paulsson magnus.paulsson@hik.se

Requires:

  • numpy (compile it linked with mkl, acml or atlas!)

For speed compile the fortran subroutines in fortran directory:

cd Inelastica/fortran
source compile.bat

UNITS: Always eV and Angstrom!

k-values always given in range [0,1.0] (or [-0.5,0.5]). They are not in reciprocal space. Instead they corresponds to the mathematical orthogonal space that is fourier transformed.

Inelastica.pyTBT.WritePDOS(fn, options, DevGF, DOS, basis)[source]¶

PDOS from the surface Green’s function from electrode calculations

NOTE! The DOS is a sum over the atoms of the unitcell. NOTE! The outfile contains the DOS divided into s,p,d,f shells. This decomposition is not perfect since polarized basis orbitals will end up in L+1 part, i.e., 6s polarized orbital = 6p
