A simple interface to evaluate electron and phonon bands on a set of points in reciprocal space.

The input file format with N points is simply:

kx(1) ky(1) kz(1) [label] kx(2) ky(2) kz(2) [label]
...
kx(N) ky(N) kz(N) [label]

Units: 1/Ang.

Phase factors defined as: exp(i k.r)

Thomas Frederiksen, March 2015
